A. C++ - Input e Output 1
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

Neste problema introdutório o teu objetivo é muito simples: deves ler um nome do input, e cumprimemtar essa pessoa!

Por exemplo, se o nome for "Pedro", deves imprimir "Bom dia Pedro!".

Input

O input consiste num nome, que é uma palavra com letras maiúsculas ou minúsculas, sem acentos. O nome não terá mais de $$$10$$$ caracteres.

Output

O output imprimido por ti deve ser uma nova linha, com o texto "Bom dia [NOME]!".

Tem cuidado para criares uma nova linha depois de imprimires o resultado. Se não o fizeres irás obter o veredicto "Wrong Answer".

Examples
Input
Pedro
Output
Bom dia Pedro!
Input
Joao
Output
Bom dia Joao!

B. Festa do Ricardo
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

É o aniversário do Ricardo. Como tal, a sua família comprou-lhe um bolo com $$$N$$$ fatias iguais para todos comerem. O Ricardo é um rapaz justo, por isso quer que cada elemento da sua família coma o mesmo número de fatias. Além disso, o Ricardo não quer dividir nenhuma das fatias, como tal cada fatia só pode ser comida por no máximo uma única pessoa. Sabendo que a família do Ricardo tem $$$M$$$ (incluindo o Ricardo) elementos, qual o máximo número de fatias que o Ricardo pode dar a cada um para que todos comam o mesmo número de fatias?

Input

Uma linha de input com os inteiros $$$N$$$ e $$$M$$$, $$$1 \leq N,M \leq 10^{15}$$$.

Output

Uma linha de output com a resposta.

Scoring

Para um conjunto de pontos valendo $$$60\%$$$ do total, é válido que $$$1 \leq N,M \leq 10^{9}$$$.

Examples
Input
8 4
Output
2
Input
8 3
Output
2

C. Pedro o trabalhador
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

O Pedro trabalha no controlo de qualidade da empresa "Caixas Azevedo e Irmãos" que, como o nome indica, fabricam todo o tipo de caixa, sendo que cada caixa tem o formato de um paralelepípedo. Hoje está-se a produzir caixas para diversos eletrodomésticos. Para que uma caixa destas passe o controlo de qualidade é importante que as suas dimensões estejam corretas. Cada caixa tem uma altura de $$$A$$$ centímetros, uma largura de $$$L$$$ centímetros e um comprimento de $$$C$$$ centímetros. Para passar o controlo de qualidade, é importante que a caixa tenha volume pelo menos $$$50$$$ centímetros cúbicos e que a altura seja pelo menos $$$3$$$ centímetros. Dadas as dimensões de uma caixa, consegues ajudar o Pedro a determinar se a caixa passa o controlo de qualidade (indicado pelo número 1 no output) ou não (indicado pelo número 0 no output)?

Input

Uma linha com os inteiros $$$A$$$, $$$L$$$ e $$$C$$$. $$$1 \leq A,L,C \leq 100$$$.

Output

Uma linha com 0 (se não passar no controlo de qualidade) ou com 1 (se passar).

Examples
Input
3 3 3
Output
0
Input
20 21 22
Output
1

D. Os 3 Irmãos
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

O David tem 3 filhos: o Nassaver, o Massavek e o Nassuer. Ele quer tirar uma fotografia de frente aos três filhos simultâneamente, de forma a que todos os irmãos apareçam na foto. Como todos os 3 filhos têm alturas diferentes, um irmão mais alto nunca pode estar à frente de um mais baixo.

Consegues ajudar o David a tirar uma boa fotografia?

Input

Uma única linha com $$$3$$$ inteiros distintos $$$a,b,c$$$, as alturas dos filhos do David. É garantido que $$$1 \leq a,b,c \leq 10^9$$$.

Output

Uma única linha, com as três alturas dos irmãos, de forma a que o primeiro número não seja mais alto que o segundo nem o terceiro, e o segundo não mais alto que o terceiro.

Example
Input
185 142 173
Output
142 173 185

E. Os Selos de Camarneiro
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

Na Camarolândia a internet ainda não foi inventada e por isso os seus habitantes são obrigados a trocar mensagens por cartas. Camarneiro, o rei da Camarolândia, gosta de colecionar os selos das cartas que recebe (e sendo o rei, já recebeu vários milhares de cartas). Na Camarolândia, todos os selos são triângulos em vez de retângulos.

Sendo tu o braço direito do rei, ele pede-te que separes a sua coleção de selos em 3 caixas: uma com os selos em forma de triângulo agudo, uma com os selos em forma de triângulo retângulo e outra com os selos em forma de triângulo obtuso.

Dica: Lembra-te do teorema de pitágoras para triângulos retângulos!

Agora experimenta "usar" o teorema de pitágoras para outros triângulos, e observa o que "falha" no teorema.

Se ainda te parece impossível resolver este problema, não hesites em contactar um dos instrutores por Discord para receberes uma nova dica.

Se porventura já sabes o que são funções trigonométricas, investiga a chamada "Lei dos Cossenos", que generaliza o Teorema de Pitágoras e ajuda a perceber a solução deste problema!

Input

Uma única linha com $$$3$$$ inteiros $$$a, b, c$$$, o comprimento dos lados do selo. É garantido que $$$1 \leq a,b,c \leq 2000$$$. É também garantido que os inteiros dados formam um triângulo possível e não-degenerado.

Output

Uma única linha com a palavra "Agudo"/"Retangulo"/"Obtuso", consoante o tipo do selo dado.

Examples
Input
4 3 5
Output
Retangulo
Input
14 11 9
Output
Agudo
Input
5 4 8
Output
Obtuso
